KPG 193 Test System

The KPG 193 (Korean Power Grid 193-bus) is a synthetic test system representing the Korean power grid, developed exclusively from open-source data to support power system research and decarbonization studies. This test system captures Korea’s unique characteristics: generation mix, renewable penetration, concentrated urban demand, and isolated grid structure.

LICENSE

The test system is released under the Open Database License (ODbL) v1.0, which allows free use, modification, and sharing while requiring attribution to OpenStreetMap contributors whose data was used in the network topology construction.

Attribution required: “Map data © OpenStreetMap contributors, licensed under ODbL 1.0”

Background

The KPG 193 was developed using clustering techniques applied to open data sources, as detailed in:

Geonho Song and Jip Kim, “KPG 193: A Synthetic Korean Power Grid Test System for Decarbonization Studies,” arXiv preprint arXiv:2411.14756, 2024.

Available at: https://arxiv.org/abs/2411.14756v2

The methodology involved:

  1. Data collection from OpenStreetMap, KEPCO reports, and municipal-level statistics
  2. Spatial clustering based on KEPCO regional office locations to reduce 230 municipalities to 193 buses
  3. Network topology extraction from OpenStreetMap power infrastructure data
  4. Demand and renewable allocation using population-weighted and capacity-based methods
  5. Validation through Unit Commitment and AC Optimal Power Flow studies over the full year

Generation Mix (2022 KPX Baseline)

The system represents Korea’s generation portfolio with realistic characteristics:

  • Coal: 46 units, ~37% of installed capacity
  • LNG (Natural Gas): 54 units, ~32% of installed capacity
  • Nuclear: 22 units, ~31% of installed capacity
  • Renewables: Solar, wind, and hydro with capacity data and hourly profiles

Temporal Coverage

  • Full year simulation: 365 days of hourly data (8,760 hours)
  • Demand profiles: Hourly load for all 193 buses
  • Renewable generation: Hourly capacity factors for solar, wind, and hydro
  • Weather data: Meteorological inputs from Korea’s LDAPS system
  • Reference solutions: Pre-computed UC decisions for validation
