Main Components

1. Header Bar

  • Application title and version
  • Current status indicator
  • Help and settings buttons

2. Solver Selector (Left)

  • Four solver types: ED, UC, DC-OPF, AC-OPF
  • Visual icons for each solver
  • Active solver highlighted

3. Configuration Panel (Center)

  • Test system selection
  • Day/hour/period inputs
  • Solver-specific options
  • Solve button

4. Output Panel (Right)

  • Two tabs: Console and Results
  • Real-time solver output
  • Solution visualization

Parameter Guide

Test System Selection

Available options:

Differences:

  • v1.2: Validated, stable, documented
  • v1.5: Latest topology, updated costs
  • Custom: Import your MATPOWER file

When to use each:

  • v1.2: Standard studies, reproducibility
  • v1.5: Latest features, current research
  • Custom: Your own network

  • Tooltip shows valid range

Horizon (UC Only)

Input: Number of hours (1-168)

Common values:

  • 24: Single day (standard)
  • 48: Two-day lookahead
  • 72: Three days (weekend)
  • 168: Full week

Computational impact:

  • 24h: ~45 seconds
  • 48h: ~2-3 minutes
  • 168h: ~30-60 minutes

Reserve Requirements (UC Only)

Typical values:

  • Spinning: (online units)
  • Supplemental: (quick-start)

Impact on results:

  • Higher reserves More units online Higher cost
  • Insufficient reserves May be infeasible

Solver Options

MIP Gap (UC only):

  • 0.00 = Exact optimal (slow)
  • 0.01 = 1% gap (standard)
  • 0.05 = 5% gap (fast, acceptable)

Time Limit:

  • Stops solver after limit
  • Returns best solution found
  • Useful for large problems

Warm Start:

  • Speeds up similar problems
  • Especially useful for UC
  • Requires compatible previous solution

Actions and Buttons

Primary Actions

[SOLVE] Button

  • Starts optimization
  • Disabled during solving
  • Changes to [CANCEL] when running

[CANCEL] Button

  • Appears during solving
  • Terminates solver
  • Returns partial results (if available)

[RESET] Button

  • Clears all configuration
  • Returns to default values
  • Confirms before resetting

Secondary Actions

[Load Configuration]

  • Load saved parameter sets
  • JSON format
  • Useful for reproducibility

[Save Configuration]

  • Save current settings
  • Includes all parameters
  • Can share with collaborators

[Export Results]

  • CSV, JSON, or GeoJSON format
  • Includes all solution data
  • Choose output location
