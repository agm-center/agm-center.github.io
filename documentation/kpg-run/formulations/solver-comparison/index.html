Selection by Application

Research Applications

Decarbonization Studies

  • Primary: UC (capture cycling costs)
  • Secondary: DC-OPF (transmission needs)
  • Validation: AC-OPF (feasibility check)

Renewable Integration

  • Primary: UC (ramping, reserves)
  • Secondary: DC-OPF (congestion from variable generation)
  • Detailed: AC-OPF (voltage impact)

Transmission Planning

  • Primary: DC-OPF (congestion, LMPs)
  • Validation: AC-OPF (true flow limits)
  • Optional: UC (temporal patterns)

Market Design

  • Primary: DC-OPF (locational pricing)
  • UC: Day-ahead commitment
  • Validation: AC-OPF (deliverability)

Operational Applications

Day-Ahead Markets

  • Stage 1: UC (commitment schedule)
  • Stage 2: DC-OPF (dispatch + LMPs)
  • Post-analysis: AC-OPF (feasibility)

Real-Time Dispatch

  • Primary: AC-OPF (must be feasible)
  • Backup: DC-OPF (if AC fails)

Congestion Management

  • Primary: DC-OPF (identify constraints)
  • Validation: AC-OPF (verify relief)

What Each Model Can Answer

Economic Dispatch (ED)

Questions ED can answer:

  • âœ“ What is the minimum possible generation cost?
  • âœ“ Which generators should run based on merit order?
  • âœ“ What is the system marginal price?
  • âœ“ How do fuel prices affect dispatch?

Questions ED cannot answer:

  • âœ— Is the dispatch AC feasible?
  • âœ— Are transmission lines overloaded?
  • âœ— What are the locational prices?
  • âœ— How should units be scheduled over time?

Unit Commitment (UC)

Questions UC can answer:

  • âœ“ Which units should be online each hour?
  • âœ“ When should units start up and shut down?
  • âœ“ What are the total cycling costs?
  • âœ“ Are reserve requirements met?
  • âœ“ How much ramping capability is needed?

Questions UC cannot answer:

  • âœ— Are there transmission constraints?
  • âœ— What are locational prices?
  • âœ— Is reactive power adequate?
  • âœ— Are voltages within limits?

DC Optimal Power Flow (DC-OPF)

Questions DC-OPF can answer:

  • âœ“ Which transmission lines are congested?
  • âœ“ What are locational marginal prices?
  • âœ“ Where should new transmission be built?
  • âœ“ How much does congestion cost?
  • âœ“ What is the optimal dispatch with network?

Questions DC-OPF cannot answer:

  • âœ— Are voltages acceptable?
  • âœ— Is reactive power sufficient?
  • âœ— What are the transmission losses?
  • âœ— How should units be scheduled over time?

Questions AC-OPF can answer:

  • âœ“ Is the dispatch truly feasible?

Best Practices

Model Selection Checklist

Ask yourself:

  1. Time horizon?

    • Single period â†’ OPF
    • Multiple periods â†’ UC

  2. Network important?

    • No â†’ ED/UC
    • Yes â†’ DC-OPF/AC-OPF

  3. Voltage critical?

    • No â†’ DC-OPF
    • Yes â†’ AC-OPF

  4. Speed requirement?

    • Fast â†’ ED/DC-OPF
    • Moderate â†’ AC-OPF
    • Can wait â†’ UC

  5. Accuracy need?

    • Approximate â†’ ED/DC-OPF
    • High â†’ AC-OPF
    • Complete â†’ UC + AC-OPF
