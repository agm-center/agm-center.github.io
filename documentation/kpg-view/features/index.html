Animation Performance

Optimization features:

  • Canvas rendering (60 fps)
  • View-based culling (only animate visible lines)
  • Throttled updates (reduce CPU)
  • Pauseable/resumable

Controls:

  • Play/Pause: Space bar
  • Speed: Slider or +/- keys
  • Reset: R key

Preset Examples

1. Capital Region Buses:

  • All buses in Area 1
  • Highlights Seoul metropolitan area
  • Useful for load center analysis

2. Nuclear Fleet:

  • All nuclear generators (22 units)
  • Typically coastal locations
  • Shows baseload generation

3. Critical Transmission:

  • Lines >90% loaded
  • Identifies bottlenecks
  • Congestion analysis

4. Renewable Generation:

  • Solar, wind, hydro sites
  • Geographic distribution
