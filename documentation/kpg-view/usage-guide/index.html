Viewing Network Elements

Toggle visibility:

Try:

  1. Uncheck “Lines” → See buses and generators only
  2. Check “Labels” → Bus IDs appear
  3. Re-check “Lines” → Complete network

Inspecting Elements

Click any element:

  • Tooltip appears with details
  • Element highlights
  • Info panel updates

Time Series Analysis

Visualize daily evolution:

In viewer:

  1. Load hour 1
  2. Take note of dispatch pattern
  3. Load hour 6 (morning ramp)
  4. Observe generation increase
  5. Continue through peak (hour 18)
  6. Create animation or composite view

Creating Presentation Views

For reports/presentations:

flowchart TB
    A[Prepare View] --> B[Apply Filters]
    B --> C[Select Key
Elements]
    C --> D[Apply Colors]
    D --> E[Add Annotations]
    E --> F[Export High-Res
Image]

Example: Coal retirement presentation

  1. Select all coal generators
  2. Color them red
  3. Show connecting transmission
  4. Add statistics overlay
  5. Export as PNG (3840×2160)
  6. Use in presentation
