Data Format Specification

Data Format Specification

Complete specification for GeoJSON files used in KPG View, including feature types, required properties, and conversion from simulation results.

GeoJSON Overview

KPG View uses the GeoJSON format for representing power system networks with geographic information.

graph TB
    A[GeoJSON File] --> B[FeatureCollection]
    
    B --> C[Feature 1:
Bus]
    B --> D[Feature 2:
Generator]
    B --> E[Feature 3:
Branch]
    B --> F[Feature N:
...]
    
    C --> G[Geometry:
Point]
    C --> H[Properties:
bus_id, voltage, ...]

Basic Structure

Feature Types

Bus Features

Geometry: Point (single coordinate)

Required properties:

PropertyTypeUnitDescription
typestring-Must be “Bus”
bus_idinteger-Unique bus identifier
base_kvnumberkVBase voltage level
latnumberdegLatitude
lngnumberdegLongitude

Optional properties:

PropertyTypeUnitDescription
load_mwnumberMWActive power load
load_mvarnumberMVArReactive power load
voltage_punumberp.u.Voltage magnitude
angle_radnumberradVoltage angle
lmpnumber$/MWhLocational marginal price
areainteger-Area/zone number
statusinteger-1=in-service, 0=out

Generator Features

Geometry: Point (at bus location)

Required properties:

PropertyTypeUnitDescription
typestring-Must be “Generator”
gen_idinteger-Unique generator identifier
bus_idinteger-Connected bus
fuel_typestring-”Coal”, “LNG”, “Nuclear”, “Solar”, “Wind”, “Hydro”
capacity_mwnumberMWMaximum capacity
latnumberdegLatitude
lngnumberdegLongitude

Optional properties:

PropertyTypeUnitDescription
output_mwnumberMWCurrent active power output
output_mvarnumberMVArCurrent reactive power output
p_min_mwnumberMWMinimum generation
cost_per_mwhnumber$/MWhMarginal cost
statusinteger-1=online, 0=offline
commitmentinteger-1=committed, 0=off (for UC results)

Branch (Line) Features

Geometry: LineString (array of coordinates)

Required properties:

PropertyTypeUnitDescription
typestring-Must be “Branch”
line_idinteger-Unique line identifier
from_businteger-Sending bus ID
to_businteger-Receiving bus ID
voltage_kvnumberkVOperating voltage

Optional properties:

PropertyTypeUnitDescription
flow_mwnumberMWActive power flow
flow_mvarnumberMVArReactive power flow
limit_mvanumberMVAThermal limit
loading_pctnumber%Percent of thermal limit
r_punumberp.u.Resistance
x_punumberp.u.Reactance
length_kmnumberkmPhysical length (estimated)
statusinteger-1=in-service, 0=out

PropertyTypeUnitDescription

Coordinate Systems

Latitude and Longitude

Format: WGS84 (EPSG:4326)

⚠️ Order matters: Longitude first, then latitude!

Valid ranges:

  • Longitude: -180 to 180 degrees
  • Latitude: -90 to 90 degrees

Korea typical:

  • Longitude: 124-132°E
  • Latitude: 33-39°N

Getting Bus Coordinates

Option 1: Use provided coordinates

Option 2: Geocode from names

Option 3: Manual positioning

  • Use geojson.io
  • Click locations on map
