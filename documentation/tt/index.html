Economic Dispatch

Definition

The Economic Dispatch (ED) module in OPEN-EGO is designed to optimize the generation of electricity in a power system while minimizing costs. It considers various constraints, including generation limits, ramp rates, and transmission constraints. This module is essential for understanding how different generation sources can be dispatched to meet demand efficiently.
