Purpose

This page documents the minimum end-to-end workflow in KPG Platform:

  1. Download the KPG test dataset
  2. Run DC OPF in KPG Run
  3. Export the solved results
  4. Visualize outputs in KPG View

Demonstration Scenario

  • Dataset: KPG193 v1.5
  • Example solve mode: DC OPF - DC Optimal Power Flow
  • Example run window: Day 1 (24 hourly snapshots)

Step 1: Download KPG TestGrid Dataset

Open the KPG TestGrid Repository and download the dataset ZIP package.

  1. Click Code
  2. Select Download ZIP
  3. Extract the ZIP to a local folder

Step 2: Load the Folder and Solve DC OPF in KPG Run

In KPG Run, configure the run directly from the extracted dataset folder.

  1. Click Change Folder and select the extracted dataset
  2. Confirm the data check message (All required and optional files found)
  3. Set Problem Type to DC OPF - DC Optimal Power Flow
  4. Set the day index and click Solve

Expected checks:

  • Solve status is successful (locally solved / optimal)
  • Result tables and charts are available in the Results tab

Step 3: Export the Solved Result Bundle

After solving, move to the Results tab and click Download ZIP.

This export contains the snapshot outputs used in KPG View.

Step 4: Open the Result in KPG View

Open KPG View and load the exported result file (ZIP or extracted snapshot folder).

  1. Click Open
  2. Select the downloaded result package
  3. Use Layers to turn feature groups on/off
  4. Use Details/Selection to inspect buses, lines, and generators

Next Step
